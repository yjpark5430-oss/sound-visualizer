클릭하면 음악이 재생되고, 음량/주파수 분석 값으로 화면 요소들이 움직이는 비주얼라이저입니다.

----


1. 기능 요약

- 오디오 로드/재생: assets/music.mp3 로드 후 클릭으로 재생 시작
- 분석 객체:
  - p5.Amplitude() (음량)
  - p5.FFT() (주파수)
- 오디오 데이터 >> 시각 매핑:
  - 음량(Amplitude) >> 원 크기 / 위치 흔들림 / 회전 속도 / 색 변화
  - 주파수(FFT) >> 방사형 선들의 길이 변화
  - 저역(bass,  low 음역대) >> 화면 아래에서 위로 차오르는 사각형(배경 레이어)
- 도형 여러 개 사용: ellipse, rect, line
- 사용자 인터랙션:
  - 클릭 / Space: 재생·일시정지
  - C: 색상 모드 전환

----


2. 조작법

- 마우스 클릭: 재생 시작 / 재생,일시정지 토글
- Space: 재생,일시정지 토글
- C: 색상 모드 전환(HSB/RGB)

처음 재생은 클릭 반드시 필요 (브라우저 정책)
